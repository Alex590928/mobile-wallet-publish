# Mobile Wallet Publish

Готовый статический проект для публикации на Vercel.

## Локальный запуск

Открой `index.html` в браузере или запусти:

```bash
python -m http.server 8080
```

## Публикация на Vercel

1. Загрузи папку в GitHub
2. Подключи репозиторий в Vercel
3. Framework Preset: Other
4. Build Command: пусто
5. Output Directory: пусто
6. Deploy
